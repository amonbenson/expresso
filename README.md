# Expresso

An open source USB &ndash; Midi &ndash; Expression pedal converter.

⚠️ Please note: This project is currently under heavy development and not yet functional. ⚠️

✔️ Supports sustain pedals of any polarity<br>
✔️ Supports any kind of three-pin analog expression pedal<br>
✔️ Pedal state can be output to both Din- or USB-Midi<br>
✔️ Can be used as a classic USB-Din-Midi adapter<br>
